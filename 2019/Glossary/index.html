    Declaration

    此为个人原创的选译词汇表，仅供参考。

    • 原载于：Glossary

    Machine Learning

    English 英文术语 Chinese 中文意译
    initialize 初始化
    normalize 标准化
    feature 特征（向量）
    outliers 异常值、异常点、孤立点
    fine-tune 微调
    softmax softmax 算法
    log loss logistic 损失
    log-space 对数空间
    train / training 训练
    test / testing 测试
    accuracy 准确度
    mAP 平均准度、平均查准率的均值
    ground truth 真实、事实、现实参照
    GPU days 在单个 GPU 上计算所需的天数

    Neural Network

    English 英文术语 Chinese 中文意译
    CNN 卷积神经网络
    DCNN 深度卷积神经网络
    FCN, fully convolutional network 全卷积网络
    R-CNN 基于区域（检测框）的 CNN
    pre-train 预训练
    forward pass 神经网络的正向传递
    BP, back-propagation 神经网络的反向传播
    SGD, stochastic gradient descent 随机梯度下降
    hyper-parameters 超参数
    FC, fully connected 全连接
    maxpooling 最大池化

    Object Detection

    English 英文术语 Chinese 中文意译
    ROI, region of interest 感兴趣的区域
    object class 目标类别、物体类别
    background class 背景类别
    region proposal 区域候选（框）
    RPN, region proposal network 区域候选（框）的神经网络
    object proposal 目标候选（框）、物体候选（框）
    object detection 目标检测、物体检测
    object detector 目标检测器、物体检测器
    bounding-box 目标检测框、边框
    bounding-box regression 目标检测框回归
    bounding-box regressor 目标检测框回归器
    multi-task loss 多任务（导向）的损失
